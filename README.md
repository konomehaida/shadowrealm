ShadowRealm

**ShadowRealm** is a text-based C++ adventure game where you explore dungeons, fight enemies, and collect crystals to unlock the Dark Citadel.

---

 How to Run

On Windows

1. Make sure you have **MinGW** or **g++** installed.

   * If not, download it from [https://winlibs.com/](https://winlibs.com/).
2. Open **PowerShell** or **CMD** in the folder where `shadowrealm.cpp` is located.
3. Compile the game:

   ```bash
   g++ -o shadowrealm shadowrealm.cpp
   ```
4. Run it:

   ```bash
   shadowrealm.exe
   ```
